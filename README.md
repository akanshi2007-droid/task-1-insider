## 📄 Pages
### Landing Page
* Entry page of the application
* Displays introduction and navigation options
### Authentication Page
* Login / authentication interface
### Dashboard
* Displays statistics using reusable **StatCard** components
* Includes sidebar navigation

## 🧩 Components
* **Navbar** – Top navigation bar
* **Sidebar** – Dashboard navigation menu
* **StatCard** – Reusable card component for displaying metrics
